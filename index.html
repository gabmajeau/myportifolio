VERIFIQ - Sneaker verificator

Web application developed as a project for a discipline at PUC Minas Poços de Caldas, aimed at verifying and certifying the authenticity of products (initial focus on sneakers), using data analysis and blockchain technology.

My function its developer the frontend, where i used HTML5, CSS and JavaScript to made the Login page and the home page, 100% functional.

The app allows users to submit photos and detailed information about products, which are then analyzed and, if approved, receive a digital certificate. This certification is recorded on a blockchain, ensuring authenticity, security, and traceability.

    There the tools used by my team:

  • HTML5, CSS3 and JavaScript to Frontend
  • Node.js, SQLite and JavaScript to Backend
  • Blockchain (for certificate registration)

On github project, is the contact of all the team!

Visual and motor stimulation table

I participated of a the project to build a visual and motor stimulation table for disabled children at ADEFIP (Association of Physically Disabled People of Poços de Caldas) in Poços de Caldas, and I can say that it was an incredible experience.

When activated by means of colored buttons, the table turns on lamps of the same color, offering valuable sensory interaction and promoting learning in a fun and inclusive way.

The table has an internal Arduino UNO and was made with basic programming, the biggest work was the infrastructure, where relays and small transformers were needed.

This project was very important for my team, it was even reported on the regional newspaper, youtube and instagram!
